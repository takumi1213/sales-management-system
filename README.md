# Sales Management System

Spring Bootで作成した顧客管理システムです。

## 機能

・顧客登録  
・顧客一覧表示  
・顧客編集  
・顧客削除

## 使用技術

- Java 21
- Spring Boot
- Spring Data JPA
- Thymeleaf
- MySQL
- Gradle

## 画面

### 顧客一覧

顧客の一覧を表示します。

### 顧客登録

顧客情報を登録できます。

## アクセスURL
http://localhost:8080/customers

## 起動方法

```bash
./gradlew bootRun
